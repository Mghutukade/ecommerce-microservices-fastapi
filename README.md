# E-Commerce Microservices with FastAPI

A backend project built to learn and implement microservices architecture using FastAPI.

## Tech Stack

- FastAPI
- Python
- PostgreSQL
- SQLAlchemy
- Docker
- JWT Authentication
- RabbitMQ (Upcoming)

## Features

- User Service
- Product Service
- Order Service
- API Gateway
- Authentication
- Event-Driven Communication

## Project Structure

```text
services/
├── user-service
├── product-service
└── order-service
```
